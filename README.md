# KOVA — Automatizaciones & IA

Sitio web de KOVA, agencia de automatizaciones e inteligencia artificial.

## Estructura

```
KOVA/
├── index.html          # Landing page principal
├── src/
│   ├── styles.css      # Estilos globales
│   └── main.js         # JavaScript (animaciones, form, nav)
├── assets/
│   └── img/            # Imágenes y recursos visuales
├── components/         # Componentes HTML reutilizables (próximamente)
└── README.md
```

## Cómo usar

Abrí `index.html` directo en el navegador, o usá Live Server en VS Code para desarrollo.

## Stack

- HTML5 semántico
- CSS vanilla (custom properties, grid, flexbox)
- JavaScript vanilla (Intersection Observer, animaciones)
